Iteration for abroad questions

We believe that the current question on working or claiming benefits abroad is not being understood by users.
So if we rewrite the question to make it clearer, then we will see a reduction in the volume of users selecting yes in error, resulting in less follow-up calls from agents and more efficient processing of claims.

We believe that NSJSA applicants who have travelled with a partner on an armed forces overseas posting are not receiving the NI credits that they are entitled to which may affect their eligibility for NSJSA.
So if we add this question to the NSJSA service, then we will see more accurate decisions on NSJSA eligibility for armed forces partners and spouses.

Options to test:

  • Option 1 - Working abroad option 1, Armed forces option 1
  • Option 2 - Working abroad option 2, Armed forces option 2
  • Option 3 - Working abroad option 3, Armed forces option 3
